## Key Things to Check

### On Overview Page:
- Service status (should be "Running" with green indicator)
- Any warning/error banners
- Connection information (port should be 18671)

### In Logs Section:
- Recent error messages
- Connection failures
- Service restarts

### In Service Settings:
- Connection information → Verify port
- IP allowlist → Should be open
- Any maintenance notifications
